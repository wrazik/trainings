Introduction

C++ Modules were accepted in C++20. They are already merged into ISO C++ Standard Draft

Suggested module file extension: cppm(clang), ixx(msvs), cc|cpp(gcc)

Introduction 

// my_module.cppm
export module math;

export int add(int fir, int sec){
    return fir + sec;
} 

// main.cpp
import math;

int main(){
   add(2000, 20);
}

Before C++20, we have term "Translation Unit"

C++20 introduces new terms:

  • module interface unit
  • module implementation unit
  • module partition
  • module interface partition
  • module implementation partition

module interface unit

A module interface unit is a module unit where the module-declaration contains the export keyword. There can be any number of these in a module.
